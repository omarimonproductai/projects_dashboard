## Setup for GitHub Pages

1. Create a new GitHub repository named `projects_dashboard`
2. Push this code to GitHub:
   ```bash
   git remote add origin https://github.com/yourusername/projects_dashboard.git
   git branch -M main
   git push -u origin main
   ```
3. Go to repository Settings → Pages
4. Select Source: Deploy from a branch
5. Choose Branch: `main`, Folder: `/ (root)`
6. Save

Your dashboard will be available at: `https://yourusername.github.io/projects_dashboard/`
